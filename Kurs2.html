Fortsättning i HTML!

Nu när vi har täckt grunderna i HTML, låt oss gå vidare och utforska några mer avancerade aspekter av HTML som kan förbättra dina webbsidor och göra dem mer dynamiska och användarvänliga.

HTML-element kan ha attribut som ger ytterligare information om elementet. Attributen placeras inom starttaggen och består av ett namn och ett värde.

För att skapa en fin och bra hemsida så behöver du mer än HTML. Detta är vart programerings språket CSS kommer in i bilden. CSS står för Cascading Style Sheets och hjälper dig ändra färg och stil på din hemsida.

Hur kopplar man ihop HTML och CSS?

Man gör detta med < link > taggen. I början av din html kod så behövs en link tagg för att koppla samman din css och html kod. För att sedan använda CSS i din HTML kod så måste du utge vissa namn till delar av din kod. Detta gör du med attributen som visas upp nedan.

Exempel på vanliga attribut:

En enkel HTML-sida består av följande grundläggande element:

  • href (för länkar)
  • src (för bilder)
  • alt (för bilder)
  • id (för att identifiera element)
  • class (för att tilldela en stil)
  • div (för att dela upp texten i olika delar och tilldela stil)
